Just a little Nuxt playground to connect up to TRPC, Supabase, and Prisma

# Stack:

FRONTEND

- Nuxt
- Vue
- Tailwind / Flowbite

TESTING

- Vitest
- Testing Library
- Playwright
- CircleCI

BACKEND

- TRPC (nuxt)
- Prisma
- Zod

DATABASE

- Supabase

CMS

- Storyblok

DEPLOY

- Vercel

TODO:

# Dev:

1. `pnpm i`
2. Fill out `.env`
3. `pnpm prisma:generate`
4. `pnpm dev`
5. Storyblok - `pnpm dev:https` in a separate terminal
